============
Installation
============


Stable release
--------------

To install auto_setupper, run this command in your terminal:

.. code-block:: console

    $ pip install auto_setupper

This is the preferred method to install auto_setupper, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for auto_setupper can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/anosillus/auto_setupper

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/anosillus/auto_setupper/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/anosillus/auto_setupper
.. _tarball: https://github.com/anosillus/auto_setupper/tarball/master
